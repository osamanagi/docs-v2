---
title: "Delete Message"
description: "Delete a message from the chat."
openapi: "/api-reference/whatsapp/whatsapp-collection.json DELETE /{instance}/delete-message"
---

You can delete messages from the chat.

so if you want to delete a message from a specific chat pass the `chatId` and `messageId` in the query parameters.

`chatId` - The unique identifier for the chat (e.g., `111111111111@c.us`)

`messageId` - The unique identifier for the message (e.g., `true_111111111111@c.us_3EB081B845A6A6E3991180`)
