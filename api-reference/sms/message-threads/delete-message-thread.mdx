---
title: "Delete Message threads"
description: "Delete a message thread and all associated messages."
openapi: "/api-reference/sms/sms-collection.json DELETE /{instance}/delete-message-thread"
---

Delete a message thread from the database and also deletes all the messages in the thread.

<Warning>
  Deleting a thread will delete all of the messages related to the thread.
</Warning>
